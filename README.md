# Getting Started

This is a simple SpringBoot app which uses [Apache Camel Timer component](https://camel.apache.org/components/3.15.x/timer-component.html) 
for periodically evoking REST API.

### Requirements
 * Java 11

### Running the app
 `./gradlew clean bootRun`
